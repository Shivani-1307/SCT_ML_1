# SCT_ML_1
Machine Learning project that predicts house prices based on square footage, bedrooms, and bathrooms using Linear Regression and an interactive Streamlit dashboard.

# House Price Prediction Dashboard

This project implements a Linear Regression model to predict house prices using the Kaggle House Prices dataset. The model uses square footage, number of bedrooms, and number of bathrooms as input features to estimate house prices.

## Features

* Interactive Streamlit dashboard
* House price prediction based on user inputs
* Actual vs Predicted price visualization
* Model evaluation using MAE and R² Score
* Dataset preview and analysis

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit
* Plotly

## Dataset

House Prices: Advanced Regression Techniques (Kaggle)

## Model

Linear Regression

## Outcome

The model predicts house prices and provides an interactive interface for users to explore predictions by adjusting house features.
